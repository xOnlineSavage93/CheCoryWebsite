# Che Cory Restaurant Website

## What is included
- Responsive one-page restaurant website
- Appetizers, entrées, and desserts
- Restaurant hours, address, and phone section
- QR code that automatically points to the live website URL
- QR code download button

## How to preview
Open `index.html` in a web browser.

## How to put it online
Upload all three website files (`index.html`, `styles.css`, and `script.js`) to a static website host such as:
- GitHub Pages
- Netlify
- Cloudflare Pages

After the site is online, visit the live website. The QR code will automatically update to the live URL. Click “Download QR Code” to save it for menus, flyers, business cards, or signs.

## Customize
Edit `index.html` to change:
- Menu items and prices
- Address
- Hours
- Phone number
- Restaurant wording
